The Library Management System is a beginner-level project built using MySQL / SQL to manage library books and student records efficiently. It allows users to perform basic operations like adding, updating, deleting, and viewing books and student borrowing records. This project helps learners practice database design, SQL queries, and CRUD operations.


---

Features

Add new books and student records.

Update existing records (books or student details).

Delete records that are no longer needed.

View all books, students, and borrowing records.

Search for books or students using queries.
